# Support

## Repository Questions

Use the repository issue tracker for:

- bugs
- documentation gaps
- workflow or CI problems
- backlog and planning discussion

## Security Issues

Use the process in `SECURITY.md` instead of public issues.

## Runtime Expectations

- Simulator is for CI, launch, and unsupported-state validation.
- Live TrueDepth capture requires a physical device with front-facing TrueDepth hardware.
